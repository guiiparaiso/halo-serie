# Halo Series

A cinematic and interactive fan website inspired by the Halo TV series.

## Features

- Futuristic UNSC-inspired interface
- Responsive design
- Animated hero section
- Interactive navigation
- Character showcase
- Timeline and statistics sections
- Image gallery
- Custom cursor effects
- Particle system
- Sound effects
- Hidden easter eggs
- Secret Spartan Protocol mini-game

## Technologies

- HTML5
- CSS3
- JavaScript (Vanilla)

## Preview

This project recreates the atmosphere of the Halo TV series through an immersive user experience inspired by science fiction interfaces.

## Installation

Clone the repository:

```bash
git clone https://github.com/SEU-USUARIO/halo-series.git
```

Open the project folder:

```bash
cd halo-series
```

Run locally by opening:

```bash
index.html
```
